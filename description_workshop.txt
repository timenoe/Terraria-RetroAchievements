Terraria tModLoader mod that integrates Terraria with RetroAchievements, a website to earn and track achievements in classic games.

Login to RetroAchievements using the chat command: [i]/ra login <user> <pass>[/i]. Once you enter a world, you can open the chat prompt by typing [i]Enter[/i]. If you don’t already have a RetroAchievements account, you can create one for free at [url=https://retroachievements.org/createaccount.php]RetroAchievements.org[/url].

[b]If applicable, reset all previously earned local in-game achievements at the start of a new playthrough using the chat command: [i]/cheevos reset all[/i][/b]

[b][u]Notice[/u][/b]
The upgrade from v1.0.21 to v1.1.0 resulted in a save file imcompatability for custom achievements. Download [url=https://github.com/timenoe/Terraria-RetroAchievements-Save-Converter/releases/download/v1/save_converter.zip]this GUI[/url] to convert your save file to the new format and keep your progress.

[u][b]New Chat Commands[/b][/u]
[list]
[*] [i]/cheevos missing <title>[/i] - Display missing elements for a tracked achievement
[*] [i]/cheevos reset all[/i] - Reset all local in-game achievements
[*] [i]/cheevos reset <title>[/i] - Reset progress for a single local achievement
[*] [i]/ra host[/i] - Get the current RA host
[*] [i]/ra login <user> <pass>[/i] - Login an RA user
[*] [i]/ra logout[/i] - Logout an RA user
[*] [i]/ra rp[/i] - Get the current RA Rich Presence
[*] [i]/ra sync[/i] - Unlock local achievements that are already unlocked on RA
[/list]
