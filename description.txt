Mod that integrates tModLoader with RetroAchievements.

Login to RetroAchievements using the chat command: "/ra login <user> <pass>". If applicable, reset all previously earned local in-game achievements at the start of a new playthrough using the chat command: "/cheevos reset all".

Chat Commands:
- /cheevos reset all - Reset all local in-game achievements
- /ra host - Get the current host
- /ra login <user> <pass> - Log in a user
- /ra rp - Get the current Rich Presence
